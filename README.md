# data-analytics-tfm

Este proyecto explora los patrones que influyen en la decisión de contratación de los clientes.

Objetivo

    Analizar los datos de campañas de marketing de una empresa Fintech para identificar y predecir los factores que hacen que un cliente contrate un depósito.

Dataset

    Source: Dataset privado proporcionado por el master (no disponible públicamente)
    Contenido: Datos crudos y procesados

Metodología

    Limpieza de datos
    Normalización de valores
    Conversión de tipos
    Arreglar datos faltantes
    Análisis de datos
    Modelado predictivo simple
    Visualización de datos

Entregables

    Documento técnico con metodología y resultados
    Entregables solicitados para cada tarea
    Presentación final para la junta ejecutiva de la Fintech

How to Use

    Abrir el notebook para ejecutar el análisis o abrir el PDF de la presentación.
